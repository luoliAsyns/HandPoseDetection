


# 手掌姿态分析系统（基于PyTorch + MediaPipe）
一个轻量级的手掌姿态分析项目，可实时检测视频/摄像头中的手掌、绘制骨骼线、评估姿态，并生成骨骼线的多视角（正/左/右/俯视）可视化视图。

## 📋 项目简介
本项目整合了目标检测、关键点估计和3D可视化技术，核心功能包括：
- 实时检测视频/摄像头中的手掌（支持多手掌）
- 自动绘制手掌骨骼线（21个关键点）
- 基于骨骼角度的手掌姿态评估（张开/握拳/半张开等）
- 生成骨骼线的正视图、左视图、右视图、俯视图

## 🚀 使用方法

### 1. 运行主程序
```python
# 直接运行（默认使用摄像头）
python main.py

# 处理本地视频文件
python main.py --video_path "你的视频路径.mp4"
```

### 2. 交互操作
| 按键 | 功能 |
|------|------|
| `q`  | 退出程序 |
| `v`  | 生成手掌骨骼多视角视图（正/左/右/俯视） |

### 3. 输出说明
- 实时窗口：显示手掌检测结果、骨骼线、姿态评估（整体姿态/评分/手指弯曲度）
- 多视角窗口：按下`v`键弹出，展示4个视角的3D骨骼线可视化

## 📌 核心功能详解

### 1. 手掌检测与骨骼绘制
- 基于MediaPipe预训练模型，检测21个手掌关键点
- 自动将归一化坐标转换为像素坐标，适配不同分辨率
- 实时绘制骨骼连接线，支持多手掌同时检测

### 2. 姿态评估
- 计算各手指弯曲角度（拇指/食指/中指/无名指/小拇指）
- 基于平均弯曲角度评估整体姿态：
  - 完全张开（平均角度>160°）
  - 半张开（120°<平均角度≤160°）
  - 半握拳（60°<平均角度≤120°）
  - 完全握拳（平均角度≤60°）
- 0-100分姿态评分（角度越大，评分越高）

### 3. 多视角可视化
- 正视图：仰角90°，方位角0°
- 左视图：仰角0°，方位角90°
- 右视图：仰角0°，方位角-90°
- 俯视图：仰角0°，方位角0°
- 放大Z轴深度信息，提升3D可视化效果
